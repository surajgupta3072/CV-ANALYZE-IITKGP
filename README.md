# Flask CV Refiner

Flask CV Refiner is a web application built with Flask that allows users to refine their CVs. It utilizes the power of OpenAI's GPT-3.5 language model to generate refined content for CVs based on user input.

## Features

- Accepts a CV file (in PDF format) as input.
- Refines the content of the CV using the GPT-3.5 language model.
- Provides an interactive web interface for users to submit their CVs and receive refined versions.
- Displays the original and refined versions of the CV for comparison.

## Prerequisites

- Python 3.7 or higher
- Flask framework
- OpenAI GPT-3.5 API credentials
